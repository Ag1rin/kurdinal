# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-01-XX

### Added
- Manual entry of Kurdish words with dynamic fields
- JSON import functionality
- JSON preview before upload
- Arweave blockchain upload for permanent storage
- Wallet management (JWK format)
- Local storage of words using SharedPreferences
- Export words to JSON file
- Words list screen with search, edit, and delete
- Transaction history
- Statistics screen with language and dialect analytics
- Duplicate word detection
- Dark mode support
- Multi-language UI support (English, Persian)
- Error handling with user-friendly messages
- Progress tracking for uploads
- Balance checking before upload
- Wallet validation

### Features
- **Manual Entry**: Enter words with pronunciation, meanings, and dialects
- **Import/Export**: JSON file support for bulk operations
- **Arweave Integration**: Permanent, immutable storage on blockchain
- **Search & Filter**: Find words quickly
- **Statistics**: View analytics about your word collection
- **Transaction History**: Track all uploads

### Technical
- Flutter 3.24.0+
- Riverpod for state management
- Material Design 3
- Cross-platform support (Windows, macOS, Linux)
